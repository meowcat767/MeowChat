# MeowChat

An experimental chatroom.


## Getting Started

### 1. Install dependencies
```
cd server
npm install
```

### 2. Set your PostgreSQL connection string
Set the `DATABASE_URL` environment variable in your shell or `.env` file.

### 3. Start the server
```
npm start
```

### 4. Open the app
Visit `http://localhost:3000` in your browser.
